# 🍔 Restaurant Food Delivery Application

A microservices-based food delivery application built with Node.js, Express, React, and MongoDB.

## 📋 Prerequisites

- Node.js (v16 or higher)
- MongoDB (v6.0 or higher)
- npm or yarn
- Git
